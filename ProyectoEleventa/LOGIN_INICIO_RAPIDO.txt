### Paso 2: Crear tabla en BD (1 minuto)
```sql
-- Ejecuta esta consulta en SQL Server
CREATE TABLE usuarios (
    id_usuario INT PRIMARY KEY IDENTITY(1,1),
    usuario VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(50) NOT NULL,
    estado BIT NOT NULL DEFAULT 1
);

-- Agregar usuario de prueba
INSERT INTO usuarios VALUES ('admin', 'admin123', 1);
```

### Paso 3: Compilar (30 segundos)
```

```


## 🚨 SI ALGO FALLA

1. **"ComboBox vacío"** → Verifica tabla usuarios existe y tiene datos
2. **"Error de compilación"** → Verifica que Program.cs esté correcto
3. **"No puedo acceder"** → Verifica credenciales: admin/admin123
4. **"Form1 no abre"** → Verifica que exista Form1 en el proyecto

→ **MÁS AYUDA**: Ver **LOGIN_GUIA_RAPIDA.md** (sección Troubleshooting)

## 🎯 PRÓXIMOS PASOS

Después de implementar el login:

1. **Mejorar seguridad**
   - Encriptar contraseñas con bcrypt
   - Bloquear tras 3 intentos fallidos
   - Registrar accesos

2. **Agregar funcionalidad**
   - "Olvide mi contraseña"
   - "Recordar usuario"
   - 2FA (Two-Factor Authentication)

3. **Integrar con sistema**
   - Guardar usuario activo en sesión
   - Mostrar nombre de usuario en Form1
   - Implementar logout
